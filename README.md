# FAQ Chatbot

## Overview

This project is a simple FAQ Chatbot developed using Python. The chatbot reads frequently asked questions from a text file and responds to user queries by finding the closest matching question.

## Features

* Text-based chatbot
* Reads FAQs from a file
* Finds closest matching question
* User-friendly interface
* Beginner-friendly Python project

## Technologies Used

* Python
* Difflib Library

## How to Run

1. Install Python.
2. Download the project files.
3. Open terminal in project folder.
4. Run:

```bash
python chatbot.py
```

5. Ask questions and get answers.

## Sample Questions

* What is Python?
* What is AI?
* What is GitHub?
* What is Machine Learning?
